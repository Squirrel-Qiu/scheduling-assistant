# scheduling-assistant
帮助创建者快速构建值班表，参与者可以根据自己的时间段灵活选择，提交至后台生成人性化值班表

### 技术栈：Go Gin + mysql，Docker 部署

小程序名称：duty（排班助手）

API文档：https://github.com/Squirrel-Qiu/scheduling-assistant/tree/master/back-end
